# 交互：alert、prompt 和 confirm


### 创建一个简单的页面

JavaScript 代码：

```js
let name = prompt("What is your name?", "");
alert(name);
```

整个页面的代码：

```html
<!DOCTYPE html>
<html>
<body>

  <script>
    'use strict';

    let name = prompt("What is your name?", "");
    alert(name);
  </script>

</body>
</html>
```
